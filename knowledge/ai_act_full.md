# EU AI Act — Full Text (placeholder)

This file is a placeholder for the **complete text** of Regulation (EU) 2024/1689 (Artificial Intelligence Act) of the European Parliament and of the Council of 13 June 2024.

**Canonical source:** https://eur-lex.europa.eu/eli/reg/2024/1689/oj/eng

The full Act runs to 113 articles, 13 annexes, and 180+ recitals. It is **not** loaded into the audit prompt — only `ai_act_relevant.md` is, to keep prompt size and cost under control. Keep this file as a reference for future expansion (e.g., when v2 adds Article 26 deployer obligations or Annex III high-risk classification).

To populate this file for v1.x or v2 work:

1. Download the canonical HTML from eur-lex via the URL above.
2. Convert HTML to Markdown (e.g., `pandoc -f html -t markdown_strict`).
3. Add section markers per Article and Annex.

For v1 the file may remain as this stub.
